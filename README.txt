29. ft_memalloc - возвращает аллоцированную память, с заданным ей размером.

30. ft_memdel - очищает область памяти и указатель на неё.

31. ft_strnew - выделяет память под строку.

32. ft_strdel - очищает строку и указатель на неё.

33. ft_strclr - заменяет все символы в строке на "\0".

34. ft_striter - исполняет функцию F к каждому символу строки

35. ft_striteri - исполняет функцию F к каждому символу строки, так же можно поменять символ по адресу, передав его индекс (i).

36. ft_strmap - Применяет функцию f к каждому символу строки, приведенному в качестве аргумента, для создания "свежей" новой строки (с malloc(3)), полученной в результате последовательного применения функции f.

37. ft_strmapi - Применяет функцию f к каждому символу строки, переданному в качестве аргумента, передавая его индекс в качестве первого аргумента для создания "свежей" новой строки (с malloc(3)), полученной в результате последовательного применения функции f.

38. ft_strequ - Лексикографическое сравнение между s1 и s2. Если 2 строки идентичны, функция возвращает 1 или 0.

39. ft_strnequ - Лексикографическое сравнение между s1 и s2 до n символов или до тех пор, пока не будет достигнут '\0'. Если 2 строки идентичны, функция возвращает 1 или 0 иначе.

40. ft_strsub - возвращает новую строку, созданная из строки, которая была аргументом, учитывая индекс начала и len.

41. ft_strjoin - возвращает строку объединенную из двух.

42. ft_strtrim - Возвращает копию строки, заданной в качестве аргумента без пробелов в начале или в конце строки.

43. ft_strsplit - Возвращает массив "свежих" строк (все заканчиваются на '\0', включая сам массив), полученный при разделение S с использованием символа С в качестве разделителя.

44. ft_itoa - Изменят число(int) в число(char).

45. ft_putchar - печатает символ.

46. ft_putstr - печатает строку.

47. ft_putendl - печатает строку с '\n'.

48. ft_putnbr - печатает число.
